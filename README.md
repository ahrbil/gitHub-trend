# gitHub-trend

### Start
1. Clone the app
  `git clone https://github.com/ahrbil/gitHub-trend.git`
  and `cd gitHub-trend` 
 2. Install depandencies using
  `yarn` or `npm install`
 3. Start the dev server 
  `yarn start` or `npm run start`
  
### Built with
1. ReactJS
2. Styled-components

fetching data from github search api
